# Projects

# BMI Calculator
This project calculates the Body Mass Index (BMI) based on the user's height and weight.  
It also shows the BMI category like:
- Underweight
- Normal Weight
- Overweight
- Obese

# Random Password Generator
This project generates random passwords using:
- Letters
- Numbers
- Special characters

Users can enter the password length and get a secure random password.

# Weather App
This project uses the OpenWeatherMap API to display live weather details of a city.

It shows:
- Temperature
- Humidity
- Weather condition
